# TE Prototypes

UX research prototypes for exploring the AI future of the SafetyCulture Template Editor.

## Files

### `Current TE.html`
A faithful single-file HTML prototype of the current Template Editor UI. Built to support UX research and concept testing.

**Features:**
- Template header with editable title and cover image placeholder
- Page/section blocks with collapsible headers, inline rename, and kebab menu
- Question rows with drag handles, response type dropdown, and Yes/No/N/A chips
- Row selection with floating action panel (left margin) and delete button (right margin)
- Add new menu (Question, Section, Page, Table) — Page option inserts a full new page block
- Response type dropdown with search, multiple choice, and other response type options
- Mobile preview panel

### `Template editor_v1.html`
Earlier prototype iteration.

## Setup

No build step required — open any `.html` file directly in a browser.
